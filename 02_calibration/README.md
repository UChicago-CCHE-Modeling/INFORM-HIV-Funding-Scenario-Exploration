# 02 — Calibration

Calibrates six HIV-model parameters to an observed HIV prevalence target using
**Simulation-Based Inference (SBI)**. This is the second stage of the analysis:
it takes a Latin-hypercube design of model simulations and learns which parameter
values are consistent with the observed prevalence.

## Method

A neural posterior estimator (`SNPE` from the `sbi` package) is trained on the
simulation set to approximate the posterior distribution of the parameters given
a summary outcome. Concretely:

1. For each simulation instance, the summary statistic is the HIV prevalence in
   the final year-category.
2. Calibration parameters are normalized to `[0, 1]` and the prevalence outcome
   is z-standardized.
3. `SNPE` is trained with a uniform prior on the normalized hypercube.
4. The trained posterior is conditioned on the observed target prevalence, and
   10,000 draws are transformed back to the native parameter scale.

### Calibrated parameters
- `transmission.probability.insertive.infected`
- `prop.casual.sex.acts`
- `prop.steady.sex.acts`
- `prep.bl.use.prop`
- `transmission.probability.receptive.infected`
- `prep.fraction.sd`

### Calibration target
Observed HIV prevalence of **29.49%** (set as `target` in the script).

## Contents

### `data/`
- **`proportion_summary.csv`** — simulated summary outcomes, one row per
  `(instance, ycat)` (1,000 instances × 20 year-categories). The calibration
  uses `mean_hiv_prevalence` in the final year-category (`ycat = 20`).
- **`parameters.csv`** — the full wide-format parameter table, one row per
  simulation instance. The calibrated parameters are a subset of its columns
  (keyed by `instance`).
- **`incidence_data.csv`** — detailed per-tick incidence/demographic counts from
  the simulations. Retained for provenance; not read by `calibration.py`.

### `script/`
- **`calibration.py`** — trains the SBI posterior, conditions on the target,
  saves the posterior samples and diagnostic plots. Run it with `script/` as the
  working directory:

  ```sh
  cd 02_calibration/script
  python calibration.py
  ```

  Requires Python with `torch`, `sbi`, `pandas`, `numpy`, `matplotlib`,
  `seaborn`, and `scipy`. A fixed random seed (`SEED = 0`) is set for
  reproducibility; note that SBI training may still vary slightly across
  hardware/library versions.

### `output/`
Generated by `calibration.py`:

| File | Description |
|------|-------------|
| `posterior_samples_mean_10000.csv` | 10,000 posterior draws on the native parameter scale |
| `posterior_marginals_mean.png`     | 1D marginal histograms of the posterior |
| `posterior_pairplot_mean.png`      | seaborn pairplot of the posterior draws |
| `prior_vs_posterior_mean.png`      | training (prior) vs posterior per parameter |
| `posterior_pairplot_kde_mean.png`  | KDE pairplot (1D densities + 2D contours) |
