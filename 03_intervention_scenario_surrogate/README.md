# 03 — Intervention Scenario Surrogate

Builds a fast Gaussian-process (GP) **surrogate model** that predicts HIV
epidemic trajectories (prevalence and incidence over years 20–30) as a function
of two intervention levers, so that arbitrary funding/scenario combinations can
be explored without re-running the expensive agent-based simulation. This is the
third and final stage of the analysis.

## Intervention levers

The surrogate maps two inputs to epidemic trajectories:

- `art.yearly.adjustment` — yearly adjustment to ART coverage
- `prep.yearly.adjustment` — yearly adjustment to PrEP coverage

Inputs are supplied on their native scale (e.g. `-0.5` = a 50% yearly
reduction) and internally mapped to `[0, 1]` for GP fitting.

## Data organization

The training data are **20,000 model simulations, run in 4 batches of 5,000 runs
each**. Each batch has its own folder under `data/`, named in ascending run
order:

| Folder | Simulation runs |
|--------|-----------------|
| `data/batch_1/` | 1 – 5,000 |
| `data/batch_2/` | 5,001 – 10,000 |
| `data/batch_3/` | 10,001 – 15,000 |
| `data/batch_4/` | 15,001 – 20,000 |

Each batch folder contains three CSVs:

- **`proportion_summary.csv`** — prevalence-style summary outcomes per
  `(instance, ycat)` (year-category); provides `mean_hiv_prevalence`.
- **`parameters.csv`** — the full wide-format parameter table, one row per
  simulation instance; supplies the two intervention levers and the six
  calibrated "checkpoint" parameters.
- **`incidence_data.csv`** — incidence outcomes per `(instance, ycat)`;
  provides `hiv_total_incidence_per_100py`.

Each simulation *instance* combines a **checkpoint** (a draw of the six
calibrated epidemic parameters — 100 checkpoints in total, 25 per batch, coming
from the stage-02 calibration posterior) with an **intervention design point**
(an `(art, prep)` adjustment pair). The post-intervention horizon is
`ycat > 21`.

## Method

For each of the 100 checkpoints:

1. Assemble the output matrix of the QOI (prevalence or incidence) over the
   intervention design points and years.
2. Standardize it and take an **SVD**, keeping enough basis components to explain
   99% of the spectrum (capped at 5).
3. Fit an independent **heteroscedastic GP** (`hetGP::mleHetGP`, Matérn-5/2) to
   each basis component's weights over the `(art, prep)` intervention space.

The 100 per-checkpoint emulators are combined into a **composite surrogate**
whose predictive draws marginalize over calibration uncertainty. Helper
functions provide absolute predictions and predictions expressed as a
difference from the no-intervention (`art = 0, prep = 0`) baseline.

## Contents

### `data/`
`batch_1/` … `batch_4/`, each with the three CSVs described above.

### `script/`
- **`surrogate.R`** — reads and aggregates the 4 batches, fits the surrogates,
  demonstrates prediction at an example scenario, and saves the fitted surrogate
  bundle. Run it with `script/` as the working directory:

  ```sh
  cd 03_intervention_scenario_surrogate/script
  Rscript surrogate.R
  ```

  Requires the R packages `data.table` and `hetGP`. A fixed seed (`set.seed(0)`)
  is set because prediction draws are stochastic.

- **`forest_plot.R`** — a downstream consumer of the saved surrogate. It loads
  `../output/surrogate.Rdata` and produces a forest plot of the mean HIV
  incidence risk ratio (scenario vs. baseline) across (ART, PrEP)
  funding-reduction scenarios, faceted into: vary ART only, vary PrEP only, and
  vary both equally. Run `surrogate.R` first so the `.Rdata` exists:

  ```sh
  cd 03_intervention_scenario_surrogate/script
  Rscript surrogate.R      # produces output/surrogate.Rdata
  Rscript forest_plot.R    # consumes it, produces the forest plot
  ```

  Requires `data.table`, `hetGP`, and `ggplot2`.

### `output/`

Generated by `surrogate.R`:

| File | Description |
|------|-------------|
| `surrogate.Rdata`        | fitted surrogates (`gp_prevalence_fit`, `gp_incidence_fit`), the `baseline_incidence_prevalence` data, and the `predict_*` helper functions |
| `pred_prevalence.png`    | example prevalence prediction at `(art, prep) = (-0.5, -0.4)` |
| `pred_incidence.png`     | example incidence prediction at the same scenario |
| `pred_incidence_diff.png`| example incidence difference from baseline at the same scenario |

Generated by `forest_plot.R`:

| File | Description |
|------|-------------|
| `forest_incidence_risk_ratio.png`         | forest plot of mean incidence risk ratio, faceted by scenario type (ART only / PrEP only / both) |
| `forest_incidence_risk_ratio_grouped.png` | single-panel version, scenarios dodged and colour-coded at each funding-reduction level |

### Using the saved surrogate
Load `surrogate.Rdata` and call the prediction helpers, e.g.:

```r
load("output/surrogate.Rdata")

# Predict the prevalence trajectory at a 50% ART / 40% PrEP yearly reduction.
samples <- predict_hiv_composite_surrogate(c(-0.5, -0.4), gp_prevalence_fit,
                                           n_samples_per_checkpoint = 50)
pred_mean <- apply(samples, c(1, 2), mean)   # mean trajectory over years 20–30
```
