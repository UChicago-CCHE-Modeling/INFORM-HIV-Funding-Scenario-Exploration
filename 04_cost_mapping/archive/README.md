# 04 — Cost Mapping (archive)

Cost-mapping outputs that are **not** among the three main-text outputs kept in
the live pipeline (`../script/run_cost_mapping.R`). The live pipeline now emits
only:

- `forest_incidence_risk_ratio_funding` (forest plot + CSV)
- `incidence_ribbon_heatmap_panel` (ribbon + heatmap panel + ribbon CSV)
- `funding_scenarios_table` (Table 1 `.tex` + CSV)

Everything else — the contour and discrete-tile heatmaps, the bar plots, and the
full grid CSV — lives here and is regenerated by `script/run_cost_mapping_extras.R`.

## Regenerate

Run with `archive/script/` as the working directory:

```sh
cd 04_cost_mapping/archive/script
Rscript run_cost_mapping_extras.R
```

It sources the still-live modules (`parameters.R`, `cost_model.R`, `outcomes.R`,
`plot_heatmap.R`) from `../../R` and the archived `plot_bar.R` from `../R`, loads
the stage-03 surrogate, and writes into `../output`. Run stage 03 first so
`surrogate.Rdata` exists.

## Contents

### `R/`
- **`plot_bar.R`** — `plot_funding_bar()`, bar plots comparing selected
  funding-reduction scenarios (incidence, relative increase, risk ratio). Only
  the archived extras script uses it.

### `script/`
- **`run_cost_mapping_extras.R`** — self-contained driver for the archived
  outputs below.
